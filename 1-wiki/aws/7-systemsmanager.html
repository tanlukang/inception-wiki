AWS Systems Manager

AWS Systems Manager

AWS Systems Manager (Systems Manager) 以前称为“ Amazon Simple Systems Manager (SSM) ”和“ Amazon EC2 Systems Manager (SSM) ”。服务“ SSM ”的原始缩写名称仍反映在各个 AWS 资源中，包括其他一些服务控制台。

功能分类

运营管理

  • Explorer
  • OpsCenter
  • CloudWatch Dashboards

  • Trusted Advisor

    AWS Trusted Advisor 的作用类似于自定义云专家，会检查您的 AWS 环境，并发现可以节省开支、提高系统性能和可靠性或帮助弥补安全漏洞的机会。AWS Trusted Advisor 提供了以下四种类别的最佳实践：成本优化、安全、容错能力和性能提升。
